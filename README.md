# 대학교 사이트 만들기

### 프로젝트 개요
Spring boot + Vue를 활용한 대학교 웹사이트입니다.
MVC 패턴을 준수합니다.

### 주요 구현 기능

* session을 이용하여 로그인 기능을 구현.
* 웹사이트 UI를 만드는데 프론트엔드 기술을 사용함.
* 계층적 아키텍처 (controller와 helper)를 사용.
* 그 밖에 대학교 사이트에 필요한 기능.

## 사용 기술

 * Spring Boot + Vue
 * 빌드 도구 : Gradle
 * 데이터베이스 : MySQL

## 참고 서적

이 프로젝트는 웹 개발 철저 입문 (최진 지음) 책을 공부한 후 대학교 사이트에 알맞게 만들어졌습니다.

## 실행 파일

데이터베이스 sql 파일 : 프로젝트 최상단 DB폴더 (skhu_create.sql, skhu_insert.sql)
